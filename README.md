> 클론코딩

* Creat: 목록 작성
  
* Read: AsyncStorage에서 데이터 읽기
  
* Delete: 목록 삭제
	
> 추가
	
* Update: 목록 수정, 클릭시 색상 변경
	
* 완료 버튼: 완료시 목록의 제일 하단 배치, 완료 해지시 목록 제일 상단 배치, 색상 변경
